# Vespera Maison

Sito web ufficiale di **Vespera Maison**, agenzia di rappresentanza commerciale nel settore dell'arredamento e del design d'interni.

🌐 **[vespera-maison.github.io](https://vespera-maison.github.io)**

---

## Struttura

```
├── index.html       # Homepage
├── aziende.html     # Pagina aziende con filtri
├── contatti.html    # Pagina contatti e modulo
└── style.css        # Foglio di stile condiviso
```

## Tecnologie

Sito statico in HTML, CSS e JavaScript vanilla. Nessuna dipendenza esterna eccetto Google Fonts. Il modulo contatti è gestito tramite [Formspree](https://formspree.io).
